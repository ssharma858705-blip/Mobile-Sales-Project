Mobile Sales Dashboard

1. Project Title / Headline
📊 Mobile Sales Analytics: Interactive Power BI Dashboard for Sales Performance Insights

2. Short Description / Purpose
An interactive Power BI dashboard designed to analyze mobile phone sales performance. It tracks key metrics like Total Sales, Quantity Sold, Transactions, and Average Price while identifying trends across cities, brands, payment methods, and time periods.

3. Tech Stack
The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Used to create interactive dashboards and visualizations

🧹 Power Query – Used for data cleaning and transformation

🧠 DAX (Data Analysis Expressions) – Used for calculated measures such as Total Sales, Total Quantity, Transactions, Average Price, MTD, and Same Period Last Year

🔗 Data Modeling – Relationships created between Sales_Data and Custom Calendar table

📁 File Format – .pbix file used for dashboard development

4. Data Source
📂 Mobile Sales dataset containing transaction-level data including:

• Transaction ID
• Date
• Brand
• Mobile Model
• Units Sold
• Price Per Unit
• Customer Name
• Customer Age
• City
• Payment Method
• Customer Ratings
• Rating Status

5. Features / Highlights

📌 Business Problem
Businesses need a clear understanding of mobile sales performance to identify top brands, customer preferences, and payment trends.

🎯 Goal of the Dashboard
To provide an interactive platform for analyzing sales performance and generating useful business insights.

📊 Key Visuals

• KPI cards showing Total Sales, Total Quantity, Transactions, and Average Price

• Map visualization showing Total Sales by City

• Monthly trend line chart showing Total Quantity by Month

• Brand-wise sales comparison table

• Pie chart showing Transactions by Payment Method

• Customer rating analysis (Good, Average, Poor)

• Mobile model sales comparison chart

• Day-wise sales analysis visualization

📈 Business Insights

• Helps identify top-performing mobile brands and models

• Shows customer payment preferences such as UPI, Debit Card, Credit Card, and Cash

• Tracks sales trends across months and different time periods

• Provides insights into customer satisfaction through ratings
